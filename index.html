Noytext
The best way to annotate your texts for ML applications

Deploy in your server a free web app to annotate your texts

See on GitHub

Noytext is better training data

Improve the quality of your annotations by sharing the URL of your web application with colleagues and partners.

Noytext is customization

Design and generate a questionnaire to know the profile of your annotators, configure the number of required annotations for each text and the application UI

Noytext is cross-platform

Deploy it on a server and your annotators will be able to access the app through any web browser. On your laptop, tablet or smartphone, Noytext has a responsive design
